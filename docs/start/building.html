Building¶

Assuming that you have all of the required components listed in the Components sections, building the system is relatively straightforward.

Setting the solder jumpers¶

Before using the interface boards for the first time, you will need to set the four solder jumpers on the microcontroller interface board, according to the temperature sensing breakout boards you intend to use.

Each solder jumper has three pads, which collectively form a switch. To use the Adafruit MAX31865 (RTD sensors) at Header 1, for example, you should solder the middle pad to the ‘RTD’ pad on the right. Likewise, for the MAX31856 (Thermocouple sensors), solder the middle pad to the ‘TC’ pad on the left. All four solder jumpers can be continuously reconfigured, as required.

It’s a good idea to use slightly longer wires for the connections between the terminals, so that the motor controller can be lifted off the interface board, without having to continuously disconnect and reconnect the wires.

Temperature breakout boards¶

The temperature breakout boards connect to the microcontroller interface board via the female 8-pin headers. In the case of the MAX31856, which has a 9-pin male header, the 9th pin (DRDY) is not used, so can be left unconnected.
